## Overview

The `help` command displays the main help message showing all available ArchiveBox commands, usage patterns, and documentation links.

## Basic Usage

```bash
archivebox help
```

```bash docker
docker compose run archivebox help
```

## What It Shows

The help command displays:

1. **Usage patterns** - How to run ArchiveBox commands
2. **Available commands** - Complete list organized by category:
   - Meta commands (help, version)
   - Setup commands (init, install, config)
   - Archive commands (add, update, remove, etc.)
3. **Documentation links** - URLs to wiki and configuration docs
4. **Current data directory** - If running inside an initialized collection
5. **Quick start hints** - Common maintenance tasks and examples

## Command Categories

Commands are organized into groups:

### Meta Commands
- `help` - Show this help message
- `version` - Show version and system info

### Setup Commands
- `init` - Initialize a new collection
- `install` - Install dependencies
- `config` - Manage configuration

### Archive Commands
- `add` - Add new URLs
- `update` - Update existing snapshots
- `remove` - Remove snapshots
- `list` - List snapshots
- `status` - Show archive health
- And more...

## Docker Usage

When running inside Docker, the help shows Docker-specific usage:

```
Docker Usage:
    # using Docker Compose:
    docker compose run archivebox [command] [...args] [--help] [--version]

    # using Docker:
    docker run -v $PWD:/data -p 8000:8000 -it archivebox/archivebox [command] [...args]
```

## Examples

### Show Help

```bash
archivebox help
```

### Docker Help

```bash
docker compose run archivebox help
```

## Getting Help for Specific Commands

Each command has its own help:

```bash
# General help
archivebox help

# Help for specific command
archivebox add --help
archivebox config --help
archivebox server --help
```
