ArchiveBox supports various other installation methods for different platforms and use cases.

<Warning>
These installation methods are contributed by external volunteers and may lag behind the official pip channel. Use Docker or pip for the most up-to-date version.
</Warning>

## Package Managers

### Arch Linux (AUR)

Install from the Arch User Repository:

```bash
yay -S archivebox
```

- **Package**: [archivebox (AUR)](https://aur.archlinux.org/packages/archivebox/)
- **Contributed by**: [@imlonghao](https://github.com/imlonghao)
- **Maintained by**: [@jasongodev](https://github.com/jasongodev)

### FreeBSD

Use the automatic setup script:

```bash
curl -fsSL 'https://get.archivebox.io' | bash
```

This uses `pkg` + `pip3` under-the-hood to install dependencies and ArchiveBox.

### Nix/NixOS

Install using the Nix package manager:

```bash
nix-env --install archivebox
```

- **Package**: [archivebox (nixpkgs)](https://github.com/NixOS/nixpkgs/blob/master/pkgs/applications/misc/archivebox/default.nix)
- **Contributed by**: [@siraben](https://github.com/siraben)

### GNU Guix

Install using GNU Guix:

```bash
guix install archivebox
```

- **Package**: [archivebox (Guix)](https://packages.guix.gnu.org/packages/archivebox/)
- **Contributed by**: [@rakino](https://github.com/rakino)

## Desktop App (Alpha)

The Electron-based desktop app provides a native application experience.

### macOS

1. Download the latest release: [ArchiveBox.app.zip](https://github.com/ArchiveBox/ArchiveBox/releases/download/v0.6.2/Electron-ArchiveBox-macOS-x64-0.6.2.app.zip)
2. Extract and move to Applications folder
3. Launch ArchiveBox.app

### Linux

Build from source (alpha):

```bash
git clone https://github.com/ArchiveBox/electron-archivebox.git
cd electron-archivebox
npm install
npm start
```

### Windows

Build from source (beta):

```bash
git clone https://github.com/ArchiveBox/electron-archivebox.git
cd electron-archivebox
npm install
npm start
```

<Info>
The desktop app is in active development. See the [Electron ArchiveBox repo](https://github.com/ArchiveBox/electron-archivebox) for more info.
</Info>

## Auto-Setup Script

The automatic setup script works on macOS and Linux:

```bash
curl -fsSL 'https://get.archivebox.io' | bash
```

<Warning>
Review the [setup.sh source code](https://github.com/ArchiveBox/ArchiveBox/blob/dev/bin/setup.sh) before running. See ["Against curl | sh"](https://docs.sweeting.me/s/against-curl-sh) for security considerations.
</Warning>

What it does:
- Detects your OS and package manager
- Installs Python, Node.js, and system dependencies
- Installs ArchiveBox via pip
- Creates a data directory
- Runs `archivebox init --setup`

## After Installation

Regardless of installation method, the usage is the same:

<Steps>
  <Step title="Create a data directory">
    ```bash
    mkdir -p ~/archivebox/data && cd ~/archivebox/data
    ```
  </Step>

  <Step title="Initialize ArchiveBox">
    ```bash
    archivebox init --setup
    ```
  </Step>

  <Step title="Add your first URL">
    ```bash
    archivebox add 'https://example.com'
    ```
  </Step>

  <Step title="Start the server">
    ```bash
    archivebox server 0.0.0.0:8000
    ```
  </Step>
</Steps>

## Troubleshooting

### Package Version is Outdated

If the package is behind the latest release:

```bash
# Upgrade with pip to get the latest version
pip3 install --upgrade archivebox

# Or use Docker for the latest stable release
docker pull archivebox/archivebox:latest
```

### Platform-Specific Issues

For platform-specific problems:

1. Check the package repository's issue tracker
2. Consult the platform's documentation
3. Try the Docker installation as a fallback
4. Ask in the [ArchiveBox community](https://github.com/ArchiveBox/ArchiveBox/wiki/Web-Archiving-Community)

### Dependencies Missing

If dependencies are missing:

```bash
# Check what's missing
archivebox version

# Install missing dependencies with your package manager
# (varies by platform)
```
