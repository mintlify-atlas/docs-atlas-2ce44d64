## Docker Compose

<Note>
  **Recommended for most users** - Docker Compose provides the easiest installation, best security, and includes all dependencies out-of-the-box.
</Note>

### Prerequisites

- Docker and Docker Compose installed ([Get Docker](https://docs.docker.com/get-docker/))

### Installation

```bash
# 1. Create a directory for ArchiveBox
mkdir -p ~/archivebox/data && cd ~/archivebox

# 2. Download the docker-compose.yml file
curl -fsSL 'https://docker-compose.archivebox.io' > docker-compose.yml

# 3. Edit the configuration (optional but recommended)
nano docker-compose.yml  # Set ADMIN_USERNAME, ADMIN_PASSWORD, etc.

# 4. Initialize ArchiveBox
docker compose run archivebox init --setup

# 5. Start the server
docker compose up
```

Open http://web.archivebox.localhost:8000 in your browser!

<Accordion title="Docker Compose Configuration Options">
  Key settings you can customize in `docker-compose.yml`:

  ```yaml
  environment:
    # Admin user (created on first run)
    - ADMIN_USERNAME=admin
    - ADMIN_PASSWORD=YourSecurePassword
    
    # Network settings
    - LISTEN_HOST=archivebox.localhost:8000
    - ALLOWED_HOSTS=*
    
    # Public access settings
    - PUBLIC_INDEX=True          # Allow viewing snapshot list
    - PUBLIC_SNAPSHOTS=True      # Allow viewing snapshot content
    - PUBLIC_ADD_VIEW=False      # Allow submitting URLs (keep False!)
    
    # Performance settings
    - TIMEOUT=60                 # Increase for slow sites
    - YTDLP_MAX_SIZE=750m        # Max video/audio file size
    
    # Privacy settings
    - SAVE_ARCHIVEDOTORG=True    # Submit to Archive.org
    - CHECK_SSL_VALIDITY=True    # Enforce SSL verification
  ```
</Accordion>

For more details, see the [Docker documentation](https://github.com/ArchiveBox/ArchiveBox/wiki/Docker).

## pip

Install ArchiveBox as a Python package using pip.

### Prerequisites

- Python 3.11 or higher (3.13+ recommended)
- Node.js 18 or higher
- Git, wget, curl

### Installation

```bash
# 1. Install ArchiveBox and core dependencies
pip install archivebox yt-dlp playwright

# 2. Install Chromium browser
playwright install --with-deps chromium

# 3. Verify installation
archivebox version

# 4. Create a data directory and initialize
mkdir -p ~/archivebox/data && cd ~/archivebox/data
archivebox init --setup

# 5. Start the server (optional)
archivebox server 0.0.0.0:8000
```

<Warning>
  After installing with pip, you may need to install additional system dependencies:
  
  **Ubuntu/Debian:**
  ```bash
  sudo apt install python3 python3-pip python3-venv git curl wget ripgrep
  ```
  
  **macOS:**
  ```bash
  brew install python node git wget curl ripgrep
  ```
</Warning>

For detailed instructions, see the [pip installation guide](https://github.com/ArchiveBox/ArchiveBox/wiki/Install#option-c-bare-metal-setup).

## apt

Install ArchiveBox via the Debian/Ubuntu package repository.

### Prerequisites

- Ubuntu, Debian, or derivative Linux distribution

### Installation

<Note>
  The apt package may lag behind the latest pip release. After installing via apt, consider upgrading with pip to get the latest version.
</Note>

See the [apt installation guide](https://github.com/ArchiveBox/ArchiveBox/wiki/Install#option-c-bare-metal-setup) for current instructions.

After installing via apt, upgrade to the latest version:

```bash
# Upgrade to latest version with pip
pip install --upgrade --ignore-installed archivebox yt-dlp playwright
playwright install --with-deps chromium

# Verify installation
archivebox version

# Initialize in a data directory
mkdir -p ~/archivebox/data && cd ~/archivebox/data
archivebox init --setup
```

## brew

Install ArchiveBox via Homebrew on macOS.

### Prerequisites

- macOS
- Homebrew installed ([Get Homebrew](https://brew.sh/))

### Installation

```bash
# 1. Add the ArchiveBox tap
brew tap archivebox/archivebox

# 2. Install ArchiveBox
brew install archivebox

# 3. Upgrade to latest version (recommended)
pip install --upgrade --ignore-installed archivebox yt-dlp playwright
playwright install --with-deps chromium

# 4. Verify installation
archivebox version

# 5. Initialize in a data directory
mkdir -p ~/archivebox/data && cd ~/archivebox/data
archivebox init --setup

# 6. Start the server (optional)
archivebox server 0.0.0.0:8000
```

For more details, see the [macOS installation guide](https://github.com/ArchiveBox/ArchiveBox/wiki/Install#option-c-bare-metal-setup).

## System Requirements

### Minimum Requirements

- **RAM**: 512 MB (1 GB+ recommended)
- **Disk**: 1 GB for application + storage for archived content
- **CPU**: Any modern CPU (single core sufficient)

### Recommended Specifications

- **RAM**: 2 GB+ for better performance
- **Disk**: 10 GB+ for application and archives
- **CPU**: Multi-core for faster parallel archiving
- **Network**: Stable internet connection

<Note>
  Storage requirements grow with your archive size. Plan for 1-50 MB per archived page depending on the extractor methods enabled and the content type.
</Note>
